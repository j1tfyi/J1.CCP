## üì¶ Development Setup

### Prerequisites
- Deno (latest version)
- Node.js and npm
- Modern web browser

### Installation Steps
1. Install Deno
   ```bash
   curl -fsSL https://deno.land/x/install/install.sh | sh
   ```

2. Clone the Repository
   ```bash
   git clone https://github.com/WhiskeySic/j1tfyi-portal.git
   cd j1tfyi-portal
   ```

3. Prepare Frontend
   ```bash
   npm install
   npm run build
   ```

4. Run Development Server
   ```bash
   deno run --allow-net --allow-read server.ts
   ```
